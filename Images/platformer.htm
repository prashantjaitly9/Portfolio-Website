Seamless Hill Platformer Game Tileset

Description

Before you Seamless Hill Platformer Game Tileset. This is a cartoon set of graphics designed to create levels in a platform game. You can create many interesting locations using it. Add the main character, opponents and get a full 2D game. The size of the tiles: 48×48.
